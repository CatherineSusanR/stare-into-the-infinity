# 🌌 Infinity Gaze

_A calming, animated breathing website to help you relax, stare into infinity, and just let go._

---

## ✨ Features

- Breathing cycle: Inhale → Hold → Exhale → Hold
- _“Relax. Breathe. Let go.”_
- Responsive layout

---

## 🛠️ Tech Stack

- React
- Framer Motion
- HTML5 Canvas
- Tailwind CSS
